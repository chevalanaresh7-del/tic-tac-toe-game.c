# Tic Tac Toe Game in C

Description

This is a simple console-based Tic Tac Toe game written in C. Two players can play the game by taking turns and choosing positions on a 3x3 grid.

Features

* Two-player mode (X and O)
* Input validation
* Win and draw detection
* Simple and clean interface

Technologies Used

* C Programming Language

How to Run

1. Compile the program:

   ```
   gcc main.c -o game
   ```
2. Run the program:

   ```
   ./game
   ```

Example Gameplay

Players take turns entering numbers from 1 to 9 to mark their positions on the board.
